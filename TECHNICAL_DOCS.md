# Техническая документация: Искренность Кофе

## 1. Обзор проекта
Проект представляет собой современный, высокопроизводительный лендинг для кофейни премиум-класса «Искренность». Сайт разработан с акцентом на эстетику, плавные анимации и удобство пользователя на всех устройствах.

## 2. Стек технологий
*   **Framework:** React 18+ (Vite)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS (библиотека утилит для дизайна)
*   **Animations:** Framer Motion (`motion/react`)
*   **Icons:** Lucide React
*   **Fonts:** Google Fonts (Space Grotesk для заголовков, Inter для текста)

## 3. Архитектура проекта
Проект следует компонентному подходу React. Основная логика и структура сосредоточены в папке `src`.

### Структура файлов:
*   `src/App.tsx`: Корневой компонент, объединяющий все секции и содержащий глобальные элементы оформления (например, анимированные кофейные зерна на фоне).
*   `src/index.css`: Глобальные стили, конфигурация темы Tailwind и импорт шрифтов.
*   `src/components/`: Директория со всеми функциональными блоками сайта.

## 4. Описание компонентов
Сайт разбит на логические модули для удобства поддержки:

1.  **Navbar.tsx**: Адаптивное навигационное меню с эффектом размытия (glassmorphism) и плавным скроллом к разделам.
2.  **Hero.tsx**: Приветственная секция с динамическим заголовком и кнопкой призыва к действию (CTA).
3.  **Specialties.tsx**: Презентация ключевых позиций (Лавандовый Латте, Искренний Чай и др.) с использованием карточек разного размера (L/S grid).
4.  **Menu.tsx**: Интерактивное меню с фильтрацией по категориям (Кофе, Чай, Десерты, Завтраки).
5.  **About.tsx & Philosophy.tsx**: Текстовые блоки, раскрывающие ценности и историю бренда.
6.  **Gallery.tsx**: Визуальная сетка с фотографиями интерьера и атмосферы.
7.  **Contact.tsx**: Форма бронирования и контактная информация с интеграцией Яндекс.Карт (или статического изображения карты).
8.  **Footer.tsx**: Подвал сайта с социальными сетями и краткой информацией.
9.  **ScrollToTop.tsx**: Утилитарная кнопка для быстрого возврата в начало страницы.

## 5. Дизайн-система и стилизация
### Цветовая палитра:
*   **Background (bg):** Глубокий темный (`#0C0A09`).
*   **Primary (brand-primary):** Благородный золотисто-кофейный (`#D97706`).
*   **Text:** Белый (`#FFFFFF`) для акцентов и серый (`#A8A29E`) для вторичного текста.

### Эффекты:
*   **Glassmorphism:** Активное использование прозрачных панелей с `backdrop-filter: blur()`.
*   **Gradients:** Мягкие переходы для текста и наложений на изображения.

## 6. Анимации
Для создания «живого» интерфейса используется `motion/react`:
*   **Entrance Animations:** Секции плавно появляются при скролле (`whileInView`).
*   **Hover Effects:** Кнопки и карточки реагируют на наведение (масштабирование, изменение цвета, поворот иконок).
*   **Parallax Background:** Кофейные зерна на заднем плане имеют медленные анимации вращения и парения.

## 7. Производительность и оптимизация
*   Использование современных форматов изображений через URL Unsplash с параметрами оптимизации (`q=80`, `w=...`).
*   Отложенная загрузка компонентов и анимации только видимых элементов.
*   Минимальный объем CSS благодаря Tailwind JIT.

## 8. Особенности деплоя (Vercel и др.)
Для обеспечения стабильной загрузки внешних изображений (Unsplash) на внешних хостингах применены следующие меры:
*   **Referrer Policy:** В `index.html` добавлен мета-тег `<meta name="referrer" content="no-referrer" />`. Это предотвращает блокировку запросов к изображениям со стороны фотостоков.
*   **referrerPolicy:** Для критических элементов `<img>` явно указан атрибут `referrerPolicy="no-referrer"`.

## 9. Инструкции по разработке
### Локальный запуск:
1. `npm install` — установка зависимостей.
2. `npm run dev` — запуск сервера разработки на `localhost:3000`.

### Сборка:
`npm run build` — создает оптимизированную версию проекта в папке `dist`.
